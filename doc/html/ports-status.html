About the Debian Installer

The debian installer is the replacement for boot-floppies, which will be used to install future versions of Debian, beginning with sarge. It has been designed to be more modular, more easy to use, and more extensible than boot floppies. It uses cdebconf, a rewrite of debconf in c, to present questions to users. It has a number of frontend modules including text, slang, and gtk.
For further information see the Debian Installer home page, or Graphical front-end status page

Status

The Status colouring has the following meaning:
  • red: This currently does not exist / does not build
  • yellow: This currently builds, but does not install (or has not been tested)
  • green: This installs (at least some of the time. Bugs may happen).
