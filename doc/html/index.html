Web pages related to debian-installer

We have moved the official d-i web pages to http://www.debian.org/devel/debian-installer/.

To get access to these files, use these commands:

      cvs -d :pserver:anonymous@cvs.debian.org:/cvs/webwml login
      cvs -d :pserver:anonymous@cvs.debian.org:/cvs/webwml co webwml/english/devel/debian-installer

Submit patches to debian-www@lists.debian.org or you ask someone with write access there.
