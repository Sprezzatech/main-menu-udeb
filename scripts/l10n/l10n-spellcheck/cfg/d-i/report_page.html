Spellchecking d-i translations

This page contains information that should help translators to find typos in po files. A set of scripts runs every day against a checkout of the SVN repository and results are made available on this page.

The idea is to gather, for each language, all the translated strings into one big file after having filtered out all the unnecessary stuff (like comments, variables and original strings) and then running aspell non-interactively in order to produce a file containing all the words not contained in the language dictionary.

Since aspell lets you use custom word lists (WL), it will be possible to reduce the list of unknown words, creating a custom WL using the following formula:

wl = common_wl + language-specific_wl

where common_wl is a list of words common to all languages (things like Lilo, Grub, FAT, ReiserFS) and language-specific_wl is a list of words not recognized by aspell.
