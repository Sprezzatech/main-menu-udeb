Translation checkout commands for d-i

First said, you don't need to checkout the repositories by hand to translate the po files. You can get all outdated files here:

http://d-i.alioth.debian.org/l10n-stats/<LEVEL>/files/<LANGUAGECODE>/
where <STAGE> is either level1, level2, level3, level4 and level5 <LANGUAGECODE> your language code like de, es, zh_CN, ...
Look there if you need the POT-templates:
http://d-i.alioth.debian.org/l10n-stats/<LEVEL>/POT/
where <STAGE> is either level1, level2, level3, level4 or level5 <LANGUAGECODE> your language code like de, es, zh_CN, ...
If you want to checkout the repositories even so, look down.

Checkout commands

Every package lists two commands. The first is for anonymous checkout, the second for Developers/people with write access to the repositories.

Debian-Installer SVN: (Browse) 

svn co svn://svn.d-i.alioth.debian.org/svn/d-i/trunk debian-installer
svn co svn+ssh://svn.d-i.alioth.debian.org/svn/d-i/trunk debian-installer
console-data: (Browse) 
svn co svn://svn.debian.org/console-data console-data
svn co svn+ssh://svn.debian.org/svn/console-data console-data
menu: (Browse) 
cvs -z3 -d:pserver:anonymous@cvs.alioth.debian.org:/cvsroot/menu co menu
CVS_RSH=ssh cvs -d USERNAME@alioth.debian.org:/cvsroot/menu co menu
popularity-contest: (Browse) 
cvs -z3 -d:pserver:anonymous@cvs.alioth.debian.org:/cvsroot/popcon co popularity-contest
export CVS_RSH=ssh;cvs -z3 -d:ext:LOGINNAME@cvs.alioth.debian.org:/cvsroot/popcon co popularity-contest
base-config: (Browse) 
svn co svn://svn.debian.org/base-config/trunk base-config
svn co svn+ssh://svn.debian.org/svn/base-config/trunk base-config
tasksel: (Browse) 
svn co svn://svn.debian.org/tasksel/trunk tasksel
svn co svn+ssh://svn.debian.org/svn/tasksel/trunk tasksel
iso-codes: (Browse)

export CVS_RSH=ssh;cvs -d:ext:LOGINNAME@haydn.debian.org:/cvsroot/pkg-isocodes co iso-codes
