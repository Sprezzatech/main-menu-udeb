Abstract

This document describes how Debian Installer (D-I) translators need to organize their work in order to have the whole Debian GNU/Linux installation process localized (l10n) in their own language.

It is aimed at being used as a reference for all D-I translators and therefore should be read before starting to work on localisations.

1. Debian Installer i18n coordination

Some members of the D-I team have taken the responsibility of the coordination of the translators and translators teams. In all documents regarding D-I i18n process, they are referred to as "the D-I i18n coordinator".

Please look at the coordination documentation to get information about the current D-I i18n coordinators.

2. Method for working on D-I translations

When starting to translate D-I, please follow all steps detailed in the next parts of this document.

2.1. Initial steps("0th level")

Several actions are needed for a new language to be supported in D-I. These will be called "Oth" level of translations.

2.1.1. First contact with D-I development team

Get in touch with the D-I i18n coordinator. Please provide your real name (the Debian Project encourages the use of real names) and possibly your GPG key (this is not mandatory but could help in the future).

2.1.2. Add early support for your language

First needed is having your language added to languagechooser (the package responsible for choosing language at the very beginning of the Debian Installer process) This will often be handled by the D-I i18n coordinator.

The D-I i18n coordinator will indicate you the ISO-639 code for your language if you don't already know it, as well as the official English name for your language. In the rest of this document, the language code will be replaced by "xx".

2.1.3. Get initial indications regarding your language

Please provide the translation for your language name (with an initial capital letter if capital letters make sense in your language) and send it to the l10n coordinator as a single UTF-8 file containing the language name in your own language. UTF-8 IS MANDATORY HERE. For instance, the French translator will send a file containing the single word: Français.

The D-I i18n coordinator will indicate you the ISO-639 code for your language if you don't already know it, as well as the official English name for your language. In the rest of this document, the language code will be replaced by "xx".
